# Icon button

A Icon button (One icon, no text).

## Props

| Props     | Type              | Required | Default | Description                                     |
| :-------- | :---------------- | :------- | :------ | :---------------------------------------------- |
| label     | string            | Yes      |         | Will be used as aria-label for a11y             |
| icon      | ReactNode         | Yes      |         |                                                 |
| HTMLProps | HTMLButtonElement | No       |         | Spreading all button props, such as onClick etc |

## Dependency

None
