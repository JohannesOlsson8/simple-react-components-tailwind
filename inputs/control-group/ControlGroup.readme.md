# Control group

Wrapping around a radio buttons or checkboxes for correct a11y and styling.

## Props

| Props    | Type                       | Required | Default | Description     |
| :------- | :------------------------- | :------- | :------ | :-------------- |
| legend   | string                     | Yes      |         | Text for legend |
| children | radio buttons / checkboxes | Yes      |         |                 |

## Dependency

None
