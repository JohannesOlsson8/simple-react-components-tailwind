# Collapse

A collapsable section.

## Props

| Props    | Type      | Required | Default | Description                                  |
| :------- | :-------- | :------- | :------ | :------------------------------------------- |
| label    | 'string'  | Yes      |         |                                              |
| children | ReactNode | No       |         | Hidden content until the section is pressed. |
